# tic-tac-toe
C Language Program: Tic Tac Toe

## Description
Playing Tic Tac Toe on your terminal.
> This program has features:
* vs Computer
* Multiplayer

## Requirements for using it
* A software (IDE) for reviewing the code
* C/C++ compiler (Recommended: GCC Compiler)
* Command Prompt or Windows Powershell

## How to play?
1. First, run on your terminal: ``gcc tictactoe.c -o tictactoe``
2. Next, run: ``./tictactoe`` 
3. Then, the board will draw. And you can follow the instructions
